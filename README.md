# Quantum-tunnelling
In this folder I would like to note down my understanding of the Quantum tunneling and why it affects the chip manufacturing. 

## 10nm 工艺制程

10nm 指的是尺度，半导体是基于硅基底，通过微纳加工制造微型化的电子元器件。10nm 是指最小可以在这个尺度上制备出基本元器件。
